SCHIIRDOR
=========

Summary
-------

* Anonymous users can ask to be administrate through a web form.
* Users must have been created in a source LDAP resource.
* Users validate their account following a link sent by mail.
* A CubicWeb user is created, that can be administrated by moderators.
* Moderators synchronized their work with a destination LDAP resource.


Description
-----------

This cube provides a way to administrate LDAP user in a CubicWeb instance.
